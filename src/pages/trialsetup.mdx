## Build File

Add the SKIE plugin to the `build.gradle.kts` file(s) that produce Kotlin/Native Xcode Frameworks.

```kotlin
plugins {
    kotlin("multiplatform")
    kotlin("native.cocoapods")
    id("co.touchlab.skie") version "0.3.3"
}
```
